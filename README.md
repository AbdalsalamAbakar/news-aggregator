# üì∞ DailyPulse | Real-Time News Aggregator

**DailyPulse** is a modern, responsive news application built with React and Tailwind CSS. It fetches real-time headlines from around the globe using the NewsAPI, allowing users to search by keywords, filter by categories, and browse through paginated results with a clean, editorial-style interface.

## ‚ú® Features

* **Real-time Data:** Fetches live news headlines using the NewsAPI REST endpoint.
* **Search Functionality:** Dynamic search allows users to find articles by specific keywords.
* **Category Filtering:** Quickly switch between Technology, Business, Sports, Science, and more.
* **Fully Responsive:** optimized for Mobile, Tablet, and Desktop with a custom-built mobile navigation drawer.
* **Advanced UI/UX:**
    * Skeleton loading states for better perceived performance.
    * Image fallback handling for broken API links.
    * Smooth "Scroll to Top" navigation on pagination.
    * Modern glassmorphism header and hover animations.
